LiveBridge API

Frontend removido. Use seu frontend em outra URL.

API disponível em:

  • /api/recordings — listar gravações
  • /api/recordings/video — stream MP4
  • /api/init-live — iniciar live
  • /api/live/hls-master.m3u8?streamName={stream} — HLS ABR (1080 / 720 / 480)
  • /hls/live/{stream}/index.m3u8 — HLS qualidade da OBS (ingest)
